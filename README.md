# Help.Drugix Data
Схема, словари и пайплайн для обновления CSV из открытых реестров РФ.

## Что делать
1) Скачайте эти файлы и добавьте в свой репозиторий `help-drugix` соблюдая структуру папок.
2) В GitHub → Settings → Secrets and variables → Actions добавьте переменные:
   - `GRLS_URL` — ссылка на выгрузку ГРЛС (CSV/ZIP).
   - `ZNVLP_URL` — ссылка на CSV с предельными ценами (ЖНВЛП).
3) Вкладка Actions → запустите `Update data (monthly)` вручную.
4) Обновлённые CSV появятся в `data/` и будут показаны в Mini App.
